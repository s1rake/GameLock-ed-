# О платформе
Gamelock(ed) — это игровая платформа в браузере с различными играми, представленными в новой форме, а также любительский проект одного человека. Не судите строго.

# Об играх (которые есть на данный момент)
🐍 Змейка — классическая змейка с таблицей лидеров, достижениями, скинами и случайными событиями:

Бонусная еда — на карте появляются 3 синих дополнительных яблока, каждое из которых прибавляет к общему рекорду за игру по 1 очку.

Радужный дождь — с неба падают золотые яблоки, каждое из которых добавляет в рекорд 1 очко + 1 золотое яблоко.

Золотая еда — все яблоки становятся золотыми и приносят в рекорд по 3 очка + 1 золотое яблоко.

Атака — вас атакуют синие круги. Ваша цель — на протяжении 20 секунд не дать им дотронуться до вашей головы, иначе вы погибнете. Через 20 секунд они останавливаются и становятся красными. В этом состоянии их можно съесть. Каждый красный кружок приносит в рекорд 4 очка.

❌⭕ Крестики-нолики — пока что без особых дополнений, классика.

💰 Кликер — кликер с улучшениями, механикой стикеров и их крафта.
